## ğŸ§  Deskripsi Singkat

Aplikasi ini memungkinkan pengguna untuk:
- Membuat akun baru (registrasi)
- Melakukan login aman menggunakan Firebase Authentication
- Mengakses dashboard pengguna dengan data real-time dari Firestore
- Melakukan logout dengan aman

Sistem dibangun menggunakan **HTML, CSS, dan JavaScript**, serta terhubung langsung dengan layanan **Firebase Cloud**.
